# WhatsApp LM Studio API - Auto Reply

## Clone the project
```shell
git clone https://github.com/xposed73/whatsapp-lmstudio-api.git
cd whatsapp-lmstudio-api
```

## Install dependencies
```shell
npm install
```

## Run the server
```shell
npm start
```
